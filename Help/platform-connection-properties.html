ConnectionProperties

The connection definition stored in ConfigEditor.DataConnections and consumed by IDMEEditor. Full detail: BeepDM/.cursor/connectionproperties/SKILL.md.

Property groups

  • Identity — name, GUID, composite layer.
  • Provider — DatabaseType, Category, DriverName, DriverVersion.
  • Endpoint / storage — host, port, database, file path, URL, delimiters.
  • Credentials & flags — consistent flags for UI grouping and lifecycle.

Next step

After building properties, run connection validation and normalization before save and open.

REST / SaaS (Web API) shape

Connectors under Connectors/ often use WebAPIConnectionProperties (base URL, headers, OAuth or API keys) instead of RDBMS host/database fields. See REST / SaaS connectors for the pattern and flagship conn-* provider pages for worked examples.
