Core project: DataSourcesPluginsCore/InfluxDBDataSourceCore/. Class InfluxDBDataSource (namespace InfluxDBDataSourceCore) implements IDataSource with InfluxDB.Client; category NOSQL, type InfluxDB.

Registration

  • DataSourceType: InfluxDB
  • Category: NOSQL

Connection

Typically requires URL, organization, bucket, and API token (store via secure connection fields; avoid hard-coding tokens). Flux queries are evaluated through the client where the datasource exposes them.

Capabilities

Time-series semantics (measurement, tags, fields, timestamps) diverge from relational CRUD; inspect InfluxDBDataSource.cs for how entities and reads map.
