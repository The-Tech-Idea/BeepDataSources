Twitter / X

Project Connectors/SocialMedia/Twitter/. TwitterDataSource : WebAPIDataSource. DataSourceType.Twitter. Reference implementation for entity endpoint maps and required filters.

Entities

Logical entities (e.g. tweets.search, users.by_id, lists.tweets) are keys in EntityEndpoints with path fragments for Twitter API v2. RequiredFilters documents mandatory query parameters per entity.

Source & IDataSource

Implementation: Connectors/SocialMedia/Twitter/TwitterDataSource.cs — TwitterDataSource : WebAPIDataSource. Entity keys such as tweets.search and users.tweets route to Twitter API v2; list operations support max_results and optional pagination_token. See IDataSource for generic entity access and paging.
