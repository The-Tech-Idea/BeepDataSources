LiteDB

Embedded NoSQL document store in a single file. Project: DataSourcesPluginsCore/LiteDBDataSourceCore. LiteDBDataSource implements IDataSource and ILocalDB.

Registration

  • DataSourceType: LiteDB
  • Category: NOSQL (AddinAttribute)

NuGet (see csproj)

LiteDB package; Beep DataManagement packages per LiteDBDataSourceCore.csproj.

Connection properties

Point FilePath / FileName at the .db (or configured) LiteDB file. Driver metadata is resolved from ConfigEditor.DataDriversClasses for LiteDB.

Local DB operations

Use ILocalDB for file lifecycle consistent with other local providers; ensure handles are released before copy/delete (see BeepDM/.cursor/localdb).
