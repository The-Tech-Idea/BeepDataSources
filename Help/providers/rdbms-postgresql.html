PostgreSQL

Core project: DataSourcesPluginsCore/PostgreDataSourceCore. Class PostgreDataSource inherits RDBSource.

Registration

  • DataSourceType: Postgre (enum spelling in models)
  • Category: RDBMS

Connection

Typical fields: host, port (5432 default), database, user, password. Set SchemaName when your metadata or queries assume a non-public schema. Normalize identifiers per your deployment (quoted vs lower-case).

Helpers

Dialect-specific DDL/DML routes through the PostgreSQL helper in BeepDM; extend there rather than hard-coding SQL in the core unless you are overriding a narrow behavior.
