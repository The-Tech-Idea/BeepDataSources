Azure Data Explorer (Kusto)

Core project: DataSourcesPluginsCore/KustoDataSource/. Class KustoDataSource (TheTechIdea.Beep.Cloud.Kusto) implements IDataSource. Add-in registers DataSourceType.Kudosity (enum spelling in models) under category CLOUD.

Registration

  • DataSourceType: Kudosity (maps to Kusto / ADX in this core)
  • Category: CLOUD

Connection & auth

Cluster URI, database, and AAD application or user credentials are typical. See Microsoft: application authentication to Azure Data Explorer.

Capabilities

KQL vs SQL surfaces differ from relational drivers; review KustoDataSource.cs for how queries and entities are exposed to Beep.
