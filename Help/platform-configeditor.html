ConfigEditor

Persisted configuration for connections, queries, mappings, drivers, and related metadata. Full detail: BeepDM/.cursor/configeditor/SKILL.md.

Managers (façade)

  • ConfigPathManager — config root layout.
  • DataConnectionManager — DataConnections CRUD.
  • QueryManager, EntityMappingManager, ComponentConfigManager, MigrationHistoryManager.

Workflow

Use editor.ConfigEditor after the editor exists. Load collections, update via façade methods (e.g. AddDataConnection, saves for queries/mappings), refresh in-memory state when other code depends on what was just persisted.

Documented datasource families

Saved DataConnections can target any registered IDataSource. Browser help for projects in this repository is phased by family on Implementations (phased); SaaS REST command catalogs: flagship conn-* index.
